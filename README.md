# graphQL demo

graphql demo 仅做参考

## 安装

* 克隆仓库

```bash
  git clone https://github.com/GraphQL-Party/graphql-demo-ts.git
  npm install
```

* 本地安装 `mongodb`

## 使用

* 启动程序

```bash
  npm start
```

* 访问 [graphiql]('http://localhost:3000/graphiql')

  http://localhost:3000/graphiql

* 访问 [voyager]('http://localhost:3000/voyager')

  http://localhost:3000/graphiql

## 其他说明

* 项目使用 ts 开发

* 后端数据源不局限于 mongoDB

* resolver 分层结构不局限于此
